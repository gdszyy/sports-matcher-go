# 联赛匹配评价规则（排除低比赛数样本）

**作者：Manus AI**

## 一、适用范围

本规则用于评估 `sports-matcher-go` 当前联赛匹配算法在 **近两年非虚拟足球联赛样本** 上的实际准确率。评价对象不是所有联赛，而是先按样本比赛数进行过滤，再对保留下来的联赛做人工复核。

## 二、最终评价规则

之所以不能把所有联赛直接纳入统计，是因为低比赛数联赛很容易受到偶然对阵、单轮赛程缺失、TS 侧暂未入库等因素干扰。对于只有 1 至 4 场比赛的联赛，即便联赛名看起来相似，也很难形成足够稳定的业务证据。因此，本次最终采用如下口径：

| 规则项 | 最终标准 |
| --- | --- |
| 统计范围 | 近两年、非虚拟、足球联赛 |
| 最小观测量 | **`event_count >= 5`** |
| 排除规则 | `event_count < 5` 的联赛**不纳入准确率统计** |
| 正确判定 | TS 命中结果与 LS 联赛为**同一业务联赛**，允许官方长短名、品牌冠名、语言差异、历史别名等**可接受映射** |
| 错误判定 | 跨国家、跨级别、跨性别、跨年龄段、跨赛事体系，或已明显落到另一条赛事线上 |
| 强校验要求 | **地区一致**，且**性别、年龄、区域、赛制形态**等显式关键词必须一致；若出现强冲突则直接否决候选 |
| 准确率定义 | **正确匹配联赛数 ÷ 纳入统计联赛数** |

## 三、强校验关键词规则

为降低泛化名称带来的误吸附，本次将“地区、性别、年龄、区域、赛制形态”提升为**强约束特征**。这意味着，后续即便名称相似，只要这些特征不一致，也不能判为可接受映射。

| 维度 | 强校验要求 | 典型示例 |
| --- | --- | --- |
| 地区 / 国家 | LS 的国家或地区与 TS 候选必须一致；若一侧为 `International`，另一侧也必须是同类国际赛事 | `England` 不能吸附到 `ASEAN`；`Poland` 不能吸附到 `Germany` |
| 性别 | `Women`、`Female`、`Girls` 与 `Men`、`Male`、`Boys` 不得互相吸附 | `U19 Women` 不能落到无性别或男子赛事 |
| 年龄 | `U23`、`U21`、`U19` 等年龄层必须逐级一致；`Youth`、`Reserve` 只能映射到同类赛事 | `U19` 不能吸附到成年国家队赛事 |
| 区域 / 分区 | `North`、`South`、`East`、`West`、`Central` 等区域分区必须一致 | `National League South` 不能映射到 `National League` 或 `National League North` |
| 赛制 / 类型 | `Cup` 与 `League` 原则上不得互吸；`5x5`、`Futsal`、`Indoor`、`Amateur` 等特殊赛制必须同类匹配 | `FA Cup` 不能仅因词面接近吸附到 `FANS Cup`；`5x5` 不能吸附到标准联赛 |
| 层级 | 对 `NAME_LOW`、`NAME_MED` 候选，如果出现明显层级冲突则直接否决 | `4 Liga` 不能吸附到 `Liga 3` |

上述关键词的维护文件已单独沉淀到 `docs/league_guard_keywords.json`，后续如需新增同义词、地区别名或赛制词，应优先更新该文件，再同步代码实现与回归评估结果。

## 四、为什么阈值选 5 场


本次没有直接选 1、2 或 3 场，是因为这类联赛太容易被“偶然撞中”或“短期缺数”影响；也没有继续提高到 8 场或 10 场，是因为那会使纳入样本量过小，难以代表整体算法表现。综合样本稳定性和覆盖面，本次将 **5 场** 作为平衡点。

| 最小比赛数阈值 | 纳入样本数 | 正确数 | 错误数 | 准确率 |
| ---: | ---: | ---: | ---: | ---: |
| 2 | 60 | 20 | 40 | 33.33% |
| 3 | 41 | 13 | 28 | 31.71% |
| 4 | 28 | 9 | 19 | 32.14% |
| **5** | **18** | **8** | **10** | **44.44%** |

从这个敏感性结果看，`event_count >= 5` 虽然缩小了样本池，但能明显减少低样本噪声对评价结果的扰动，因此更适合作为当前阶段的正式评估口径。

## 五、按最终规则计算的本次算法准确率

基于上述最终标准，当前算法在可纳入统计的联赛样本上，结果如下：

| 指标 | 数值 |
| --- | ---: |
| 最小比赛数阈值 | **5** |
| 纳入统计联赛数 | **18** |
| 正确匹配数 | **8** |
| 错误匹配数 | **10** |
| **最终准确率** | **44.44%** |

这意味着：

> **在“排除低比赛数联赛、只统计 `event_count >= 5` 的近两年非虚拟足球联赛”这一最终标准下，当前算法的联赛匹配准确率为 44.44%。**

## 六、当前正确样本与典型错误样本

从人工复核结果看，当前算法对标准联赛名、同国直观命名的命中相对稳定，但对泛化词、层级名、年龄段和非标准小型赛事仍然存在明显误配。

### 6.1 正确样本（节选）

| LS 联赛 | 国家/地区 | TS 命中 | 说明 |
| --- | --- | --- | --- |
| Serie A | Brazil | Brazilian Serie A | 标准联赛名，国家上下文清晰 |
| Primera Nacional | Argentina | ARG Primera Nacional | 国家与联赛层级一致 |
| Super League | Greece | Greek Super League | 标准命名，语义一致 |
| U19 Liga | Serbia | Serbian U19 League | 年龄段一致，可接受映射 |
| Suomen Cup | Finland | Finland Suomen Cup | 官方长短名差异，可接受映射 |
| Liga NB III | Hungary | Hungary Nemzeti Bajnoksug III | 本地语与英文转写差异，可接受映射 |
| Serie A | Italy | Italian Serie A | 国家与赛事体系一致 |

### 6.2 典型错误样本（节选）

## 七、工程含义

这个结果说明，当前算法如果不做额外治理，仍然会在“泛化联赛名、层级名、品牌名、年龄段赛事”上发生较高比例的误配。后续如果要提高准确率，最优先的改进方向仍然是：

| 优先级 | 建议 |
| --- | --- |
| P0 | 为高频泛化名与层级名补充白名单/黑名单和精确别名映射 |
| P0 | 对 `NAME_LOW` 与 `NAME_MED` 的跨国家、跨级别候选增加更强否决条件 |
| P1 | 对年龄段、女子赛事、分区联赛单独建规则或特征 |
| P1 | 在 TS 比赛覆盖充分时，再引入比赛层反向确认做高精度补强 |
